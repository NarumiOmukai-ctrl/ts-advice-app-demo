# TypeScript × 公開API 日本語翻訳アプリ

このリポジトリは、TypeScript を使用して公開 API（Advice Slip API）から  
ランダムな「ひと言アドバイス」を取得し、日本語に翻訳して表示する  
シンプルな Web アプリのサンプルです。

Technocrats Japan 面接提出用の成果物として作成しました。

---

## 🚀 主な技術スタック

- **TypeScript**
- **JavaScript（コンパイル後）**
- **HTML / CSS**
- **Fetch API**
- **MyMemory Translation API**

---

## 🔧 機能

- 「アドバイスを取得する」ボタンで公開 API からデータを取得
- 取得した英語アドバイスを日本語へ自動翻訳
- アドバイス内容・翻訳文・アドバイスID を表示
- エラー発生時のメッセージ表示

---
